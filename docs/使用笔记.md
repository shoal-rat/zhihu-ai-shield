# 使用笔记

## 推荐启动方式

第一次使用建议先保持默认“改写成占位块”。这样即使命中错了，也能点“临时展开”查看原内容，再点“加入白名单”纠正。

等你确认阈值适合自己的阅读偏好后，再切换成“折叠”或“直接隐藏”。

## 阈值怎么调

- 误伤多：把屏蔽阈值从 `72%` 调到 `80%` 或更高
- 漏掉多：把屏蔽阈值调到 `65%`
- 小模型乱判：提高“最低置信度”
- 页面慢：关闭“读取作者近作”或把近作篇数调到 `1`

## 名单怎么写

白名单和黑名单都支持三种写法：

```text
作者名称
作者 slug
https://www.zhihu.com/people/example
```

作者 slug 是个人主页链接里的最后一段。

## 适合的模型

这个任务不是写长文，更像轻量分类。小模型通常已经够用。推荐先试本地小模型，如果误判明显，再换更强模型。

只要接口兼容 OpenAI Chat Completions，理论上都能接。
