# 隐私说明

## 本地保存什么

扩展会在浏览器存储中保存：

- 设置项
- 白名单和黑名单
- 作者判断缓存
- 最近判断记录
- 简单统计数字

这些数据默认留在你的浏览器里。

## 会发送什么

当你启用模型判断时，扩展会把以下样本发送到你配置的模型接口：

- 当前卡片标题和摘要
- 作者主页或近作页中提取到的文本片段
- 作者名称和作者链接

扩展不会主动发送 Cookie、账号密码或完整浏览历史。请求模型接口时是否会被服务端记录，取决于你配置的模型服务。

## 推荐配置

最稳妥的配置是本机模型服务，例如：

```text
http://127.0.0.1:11434/v1/chat/completions
```

如果使用远程 API，请确认服务商的数据保留策略，并避免把敏感页面加入扫描范围。

## 权限说明

扩展请求的主要权限：

- `storage`：保存设置、名单和缓存
- `https://www.zhihu.com/*`：读取知乎页面内容
- `https://zhuanlan.zhihu.com/*`：读取知乎专栏内容
- 本地和常见模型接口地址：调用你配置的分类模型

如果你要使用其他远程模型域名，需要把域名加入 `manifest.json` 的 `host_permissions`。
