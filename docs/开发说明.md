# 开发说明

## 技术栈

- Manifest V3
- 原生 JavaScript
- 原生 CSS
- Node 内置测试框架

没有构建步骤，也没有运行时依赖。这样做的目的很简单：下载后能直接加载，改代码后刷新扩展就能试。

## 本地检查

```bash
node scripts/check-manifest.mjs
node --test tests/*.test.mjs
```

## 核心模块

- `src/content/content.js`：在知乎页面找内容卡片，提取作者与文本，展示屏蔽 UI
- `src/background.js`：负责作者去重、缓存、抓近作、调用模型、统计和名单
- `src/shared/classifier.js`：提示词、模型 JSON 解析、规则兜底、决策融合
- `src/shared/zhihu.js`：知乎链接规范化、作者 slug 提取、HTML 样本提取

## 调试方式

1. 在浏览器扩展页加载本目录。
2. 打开扩展详情页。
3. 点击 Service Worker 查看后台日志。
4. 在知乎页面打开 DevTools 查看 content script 行为。
5. 修改代码后，在扩展管理页点击刷新。
